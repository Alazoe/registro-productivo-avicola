# 🐔 Registro Productivo Avícola

Sistema de registro diario de producción para gallinas ponedoras, desarrollado en Google Apps Script con interfaz web optimizada para dispositivos móviles.

**Autor:** Andrés Lazo Escobar, Médico Veterinario · avivet.cl

---

## 📁 Estructura del Repositorio

Registro-productivo-avicola/
├── src/                          ← Código base (común a todas las avícolas)
│   ├── Code.gs                   ← Lógica del servidor
│   └── index.html                ← Interfaz web móvil
├── avicolas/                     ← Una carpeta por productor
│   ├── rincon-de-chincolco/
│   ├── avicola-clarita/
│   ├── avicola-gh/
│   ├── reinhard/
│   └── praderas-de-ranco/        ← Variante con flujo diferente
└── docs/
└── como-publicar.md
