# 🌐 Cómo publicar el script como Web App

## Pasos para cada avícola nueva

1. Abre el Google Sheet de la avícola
2. Ve a **Extensiones → Apps Script**
3. Pega Code.gs y el archivo index.html
4. En el menú superior haz clic en **Implementar → Nueva implementación**
5. Tipo: **Aplicación web**
6. Ejecutar como: **Yo (tu cuenta)**
7. Quién tiene acceso: **Cualquier usuario**
8. Haz clic en **Implementar**
9. Copia la URL generada y envíasela al productor

## Actualizar una implementación existente

1. Haz tus cambios en el código
2. **Implementar → Gestionar implementaciones**
3. Haz clic en el lápiz ✏️ de la implementación activa
4. Versión: **Nueva versión**
5. **Implementar**

> ⚠️ Si solo guardas el código sin hacer una nueva versión, los cambios NO se verán en la URL publicada.

## Agregar a pantalla de inicio (iPhone / Android)

**iPhone:**
1. Abre la URL en Safari
2. Toca el botón compartir
3. Selecciona "Agregar a pantalla de inicio"

**Android:**
1. Abre la URL en Chrome
2. Toca el menú (tres puntos)
3. Selecciona "Agregar a pantalla de inicio"
